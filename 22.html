题目：

The
? 1 ? 2 ? … ? n = k problem 

 

The
problem

Given the following formula, one can set operators ‘+’ or ‘-‘
instead of each ‘?’, in order to obtain a given k
? 1 ? 2 ? … ? n = k

For example: to obtain k =
12 , the expression to be used will be:
- 1 + 2 + 3 + 4 + 5 + 6 – 7 = 12 
with n = 7

 

The
Input

The first line is the number of test cases, followed by a blank
line.

Each test case of the input contains integer k
(0<=|k|<=1000000000).

Each test case will be separated by a single line.

The
Output

For each test case, your program should print the minimal possible
n (1<=n) to obtain k with the above formula.

Print a blank line between the outputs for two consecutive test
cases.

Sample
Input

 

2 

 12 

 -3646397

 

Sample
Output

 

7 

 2701




代码：




#include<iostream>

#include<fstream>

#include<cmath>

using namespace std;

int main()

{

// freopen("test.txt","r",stdin);

 int ncase;

 cin>>ncase;

 while(ncase--)

 {

 long k=0;

 cin>>k;

 if(k==0){

 cout<<"3"<<endl;

 }

 if(k<0)k=-k;

 long tmp=(sqrt(1+8*k)-1)/2;

 long sum=(1+tmp)*tmp/2;

 while(1)

 {

 if(sum>=k&&(sum-k)%2==0)break;

 tmp++;

 sum+=tmp;

 }

 if(0!=k)

 cout<<tmp<<endl;

 if(ncase)cout<<endl;

 }

 return 0;

}
