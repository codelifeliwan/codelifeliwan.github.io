*****read
read [-pt] variable
-p : 后面接提示符   
-t:后面接等待的秒数

eg:read -p “please input something:” -t 5 name

*****declare/typeset  变量声明
declare [-aixr] variable
-a:数组类型  
-i:整型数字类型  -x:变成环境变量（同export) -r:不可更改
变量类型默认为字符串
declare +x variable “+”进行取消操作
declare -p variable “-p”可以单独列出变量类型
数组读取直接${数组} 

****命令执行
“；”分号前语句执行完立即执行下一条语句
”$?”命令回传码
cmd1&&cmd2
cmd1执行完毕正确执行则执行cmd2
cmd1||cmd2 cmd1执行完毕为错误则执行cmd2
命令执行是一个接一个的，若要使用判断则
commond1&&commond2||commong3
&&与||顺序不可颠倒

*******************************************************************************

******管道命令（pipe)  界定符号：“|”
（1）管道命令只会处理standard output,standard error output不会处理。
（2） 必须能够接收来自前一命令的数据成为标准输入处理。

******选取：
（1）cut 对一行里面数据进行分析。
   #cut -d ‘分隔字符’ -f fields
(取出f段）  -c 字符范围，以字符为单位取出
 (2) grep 分析一行信息，取关键字，若存在关键字，将一整行打印出
   # grep [-acinv] [–color=quto]
‘查找字符串’ filename

*******排序：sort,wc（查找多少行，字符等）,uniq(去重）

*******双向重定型 tee
#tee [-a]
file   
-a:以累加方式，将数据加入到file中
让标准输出转存一份到文件并讲同样数据送到屏幕去处理

*******字符转换命令
（1）tr 删除或者文字替换
   #tr [-ds]
set1   
-d:删除信息中set1字符串  -s:替换掉重复字符
   eg:# last | tr
‘[a-z]’  ‘[A-Z]’ 将小写变成大写
（2）# col [-xb]  
-x:将tab键转换成对等的空格键   
-b:文字内有时，保留最后接的那个字符 
    cat -A
显示所有的特殊按键
（3）join将两个文件中有相同数据的一行加在一起。
  #join [-ti12] file1
file2  
-t:默认以空格符分隔数据，对比第一个字段的数据，相同则两个数据连城一行
   注意之前sort
（4）paste [-d] file1(-:标准输入） file2 直接将两行连接在一起，中间以tab分隔
（5）expand [-t] file -t后面接数字，一个tab键可以转换成的空格数
（6）split 切割，依据所需文件大小切割文件
     
#split [-bl] file PREFIX  
-b:后接可切割的文件大小   -l:以行切割
     
若需将切割文件合并成一个文件只需要用数据流重定向即可。
（7）xargs 参数代换。。。。（不是很了解）产生某个命令的参数===》很多命令不支持管道命令（ls)，可以通过
   xargs提供该命令标准输入
  eg:find /sbin -perm +7000 |ls -1
//找出具有特殊权限的文件名，并详细列出
   
但并无法达到，只是输出了root所在目录下文件，由于ls不是管道命令原因
    可以修改成:find
/sbin -perm +7000 |xargs ls -1
(8) 减号- 
管道命令中，经常用到前一stdout作为这次stdin,某些命令需要用到文件名，这是这些stdout/stdin用-代替
    eg:#tar -cvf
– /home | tar -xvf – (tar -xvf – 后一个-是取用前一个命令的stdout)
