18、删除Ubuntu终端的版权说明

当登录一个Ubuntu虚拟终端（譬如用putty）的时候，通常会看到一个版权说明。

说明“Ubuntu是一个自由软件，但不承担任何责任”等等。如果你非常不喜欢每次登录都看到这些声明，你可以删除在Ubuntu主机上面删除它。这些内容存储在“/etc/motd”文件里面。所以可以：

sudo rm /etc/motd

sudo touch /etc/motd

就不在显示版本说明了。

当然，你可以修改这个文件的内容，输入：

sudo gedit /etc/motd

输入自定义的内容保存。下次用户登录，就会看到你的自定义了。


19、加快“Wubi安装方式的Ubuntu”的运行速度

Wubi方式安装的Ubuntu，它的文件系统在一个非常大的Windows文件里面，譬如是c:ubuntudisksroot.disk，交换分区在另外一个文件，譬如c:ubuntudisksswap.disk.

显而易见，如果在Windows里面整理一些硬盘碎片将加快Ubuntu系统的速度。当然，在安装Ubuntu以前整理磁盘碎片，更是一个绝佳的主意。。。


20、建立一个可以自动安装软件的网络链接或者Email链接

有的时候，你帮助别人解决问题需要安装一些软件。但是可能你并不能接触到Ubuntu系统。

一个解决方案就是建立一个网络链接或者Email链接。

现在，编辑一个网页。在网页里面建立一个链接，在链接地址里面输入“apt：packagename”，其中packagename是软件包的名称。

譬如，你希望告诉别人安装thunar文件管理器，可以编写一个Html格式邮件，在右键里面插入一个链接，链接的Url输入“apt:thunar”就可以了。当邮件接收人受到邮件以后，单击这个链接，Ubuntu就会自动提示安装这个软件。


21-让字体更加好看

多数字体的设计者都会对字体进行安装他们的感观进行微调。但是Ubuntu使用一个叫“autohiting”的系统进行调节，而忽略了这些东西。当然，这套系统工作的非常好，让Ubuntu看起来更加顺眼。但是，如果你还想进行“字节级别”的微调，就必须自己调节。

打开一个命令行窗口，输入：
sudo dpkg-reconfigure fontconfig-config

在终端安装时用光标，从菜单中选择“Native”，会出现下一个屏幕，是否开启subpixel渲染（对于LCD屏幕效果比较好，建立选择Automatic），进入下一个屏幕，是否开启图形字体渲染，选择是，就可以完成了。

配置完成以后，需要更新一些系统：
sudo dpkg-reconfigure fontcofig
sudo dpkg-reconfigure defoma
现在注销并重新登录，就会看到调节效果了。


22-加快更新的下载速度

每当Ubuntu新版本发布的时候，Ubuntu的更新服务器就会变得有些缓慢。这个时候，你可以根据自己的地理位置，选择另外一个更新服务器（在世界各地都有同主服务器同步的Ubuntu更新源）。

选择“系统”-》“管理”-》“软件源”，从“下载”下拉框里面选择“其他服务器”，会出现一个Ubuntu服务器列表，选择一个自己需要的即可（当然还有一个更好的“测试软件源的速度”的功能）。更换以后，你需要重新载入一些软件列表。

不要担心一个你从来没有听说过的服务器的安全状况，所有的Ubuntu软件包都有数字签名，所以篡改软件在技术上是不可能的。


23-减慢“触摸板”的拖动速度

如果你有一个netbook，可能在Windows系统里面有一个很好用的触摸板功能-edge
scroll。这个功能在触摸板的右手边，他会在屏幕上面显示一个虚拟的滚动条，当手指上下移动，当前窗口会自动上下滚动。
你可能意识到可以在Ubuntu也可以使用这个功能（在“系统”-》“首选项”-》鼠标里面）。但是我发现有一个问题，就是滚动的速度太快了，轻轻一点就会把一个网页从开头滚动到了末尾。

解决这个问题，需要在xorg.conf里面配置。步骤如下：
1、打开xorg.conf:
gksu gedit /etc/X11/xorg.conf
2、找到下面的两行：
Secion “InputDevice”
Driver “synaptics”
在这两行的下面添加一行：
Option “VertScrollDelta” 50
3、现在关闭使用程序，用Ctrl+Alt+BackSpace重新启动X Server，登录进去就会看到效果了。
如果你发现滚动太慢，可以把50改为25或者更小，数值越小则触摸板的敏感度越高。
想让Firefox拖动和滚动的行数少一些，可以在Firefox地址栏里面输入“about:config”（会出现一个警告，不要管它）,找到“mousewheel.withnokey.sysnumlines”项，把它更改为false。

想加快Firefox的滚动速度，输入“mousewheel.withnokey.numlines”，把它的值改为一个比默认的1更大的数值。为了是滚动更平滑，选择Firefox菜单的“编辑”-》首选项，选择“高级选项”，勾选“使用平滑滚动”


24-确保在Ubuntu总是可以访问Windows分区

有的时候，你会发现在Ubuntu下面看不到Windows分区，提示错误信息-“无法挂载卷”。这个问题的原因可能是因为Windows崩溃或者没有正常关机造成了。如果Windows没有正常关机，Ubuntu就会拒绝加载Windows分区。如果Windows正常关机了，Ubuntu还是无法加载，你可以在Windows里面使用“chkdsk”命令来检测一下Windows分区。



25-提高终端的视觉感观


更改字体，可以点击“编辑”-》“当前配置”，去掉勾选“使用系统的等宽字体”。从“字体”下拉列表里面选择其他字体，或者更改字体的大小（我发现8pt是一个好的选择）。

通常来说，不是所有的字体GNOME 终端都可以使用。

更改颜色，可以点击“编辑”-》“当前配置”，去掉勾选“使用系统主题中的颜色”。然后自己配置就可以了。




如果想增加可视面积，可以可以点击“编辑”-》“当前配置”，去掉勾选“默认在新终端显示菜单栏”选项。
