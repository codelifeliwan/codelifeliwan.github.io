事实上，这个编译是不通过的，因为泛型类型不同，其次有一种只读方法，就是只能读取数据不能修改数据，那么用？来解决：


package fanxing;

private T x;


private T y;

x=a;


y=b;


}

this.x=x;


}


public void setY(T y){


this.y=y;


}


public T getX(){


return x;


}


public T getY(){


return y;


}


}

 


public class Test001{


public static void main(String args[]){


Point<Float> a=new Point<Float>(23.45f,23.54F);


System.out.println(a.getX()+”  ,   “+a.getY());

}

 


static void fun(Point<?> a){


System.out.println(a.getX());


}


}

这里只能对数据读出而不能写入数据。

 

 


受限泛型：

给泛型设置上限和下限：

设置上限：

·声明对象：类名称<? extends 类>对象名称

·定义类：类名称<泛型标识 extends 类>

设置下限：

·声明对象：类名称<? super 类>对象名称

·定义类：类名称<泛型标识 extends 类>

泛型的其他应用：

泛型接口：

定义如下：


package fanxing;

 


interface FanxingTest001<T> {


T fun();


}


使用此接口的方法大体有两种：


1、在子类也声明泛型


package fanxing;

 


interface FanxingTest001<T> {


T fun();


}

 


class Test<T> implements FanxingTest001{


private T a;


public T fun(){


return a;


}


}


2、直接在定义的时候说明泛型类型：


package fanxing;

 


interface FanxingTest001<T> {


T fun();


}

 


class Test implements FanxingTest001<Integer>{


private int a;


public Integer fun(){


return a;


}


}

 


泛型方法：

定义格式如下：

访问权限 <泛型标识>泛型标识 方法名称 (泛型标识 参数名…)

例如：


class Demo{


public <T> T fun(T a)


{


return a;


}


}


使用方法如下所示：
