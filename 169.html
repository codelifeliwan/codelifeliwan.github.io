对象数组：


使用对象数组的时候一定要记住：用new来申请内存，因为在使用的时候得先申请栈内存，再申请堆内存的。如下：

public class Main {

   
public static void main(String args[]){

      
Test t[]=new Test[3];          
//分配栈内存

      
t[0]=new Test(“Hello”);        
//分配堆内存

      
t[1]=new Test(“World”);        
//分配堆内存

      
t[2]=new Test(“!”);            
//分配堆内存

   
}

}

 

class Test{

   
String s;

   
public Test(String a){

      
s=a;

   
}

}
