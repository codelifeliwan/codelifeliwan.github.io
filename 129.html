Given two sequences of characters, print the length of the longest
common subsequence of both sequences. For example, the longest
common subsequence of the following two sequences:

abcdgh
aedfhr

is adh of length 3.

Input consists of pairs of lines. The first line of a pair
contains the first string and the second line contains the second
string. Each string is on a separate line and consists of at most
1,000 characters

For each subsequent pair of input lines, output a line
containing one integer number which satisfies the criteria stated
above.

Sample input

a1b2c3d4e

zz1yy2xx3ww4vv

abcdgh

aedfhr

abcdefghijklmnopqrstuvwxyz

a0b0c0d0e0f0g0h0i0j0k0l0m0n0o0p0q0r0s0t0u0v0w0x0y0z0

abcdefghijklmnzyxwvutsrqpo

opqrstuvwxyzabcdefghijklmn

Output for the sample input

4 

3 

26 

14 




代码：




#include<iostream>

#include<cstring>

#include<algorithm>

#include<fstream>

#define maxn 1005

using namespace std;

char S1[maxn];

char S2[maxn];

int map[maxn][maxn];




int dp(){

 int len1=strlen(S1);

 int len2=strlen(S2);

 memset(map,0,sizeof(map));

 int m=0;

 for(int i=1;i<=len1;i++){

 for(int j=1;j<=len2;j++){

 if(S1[i-1]==S2[j-1])map[i][j]=map[i-1][j-1]+1;

 else map[i][j]=max(map[i][j-1],map[i-1][j]);

 if(map[i][j]>m)m=map[i][j];

 }

 }

 return m;

}







int main()

{

// freopen("test.txt","r",stdin);

 while(cin.getline(S1,1000)){

 cin.getline(S2,1000);

 int re=dp();

 cout<<re<<endl;

 }

 return 0;

}
