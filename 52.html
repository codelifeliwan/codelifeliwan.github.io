ubuntu11.04谷歌地球乱码问题解决

Posted on 10 六月, 2011 by wanli — No Comments ↓

在终端运行下面代码：

cd /opt/google/earth/free/
sudo rm libQtCore.so.4 libQtGui.so.4
libQtNetwork.so.4 libQtWebKit.so.4
sudo apt-get install libqt4-core libqt4-gui
libqt4-network libqt4-webkit libfreeimage3
修改文件：
sudo gedit
/opt/google/earth/free/googleearth
在（最后一行）：
LD_LIBRARY_PATH=.:$LD_LIBRARY_PATH
./googleearth-bin “$@”
前面加上：
export
LD_PRELOAD=/usr/lib/libfreeimage.so.3
然后保存。
接着从新打开谷歌地球就可以了。
