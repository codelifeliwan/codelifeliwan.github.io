其他的基本和C相同，下面讲函数：


C++里面的函数基本也是跟C一样，不过在C++里面加了几个而已，在C++里面讲函数的时候需要讲一个传值的方式：传引用。


什么是引用呢？？引用就是给变量起了个别名，用符号&表示，其实在同一函数里面和变量没有什么区别，但是在调用函数的时候在C里面是用指针改变变量的值，在C++里面不用传地址，传引用就可以，引用自己不占内存空间，就像变量的本身一样，不过就是给变量本身又多起了个名字。看下面交换变量值的例子：


#include<iostream>


using namespace std;


void swap(int &a,int &b)


{


int c;


c=a;


a=b;


b=c;


}


int main()


{


int a=2,b=3;


int &c=a;


cout<<a<<”   “<<b<<endl;


swap(a,b);


cout<<a<<”   “<<b<<endl;


c=100;


cout<<a<<endl;


return 0;


}

C++里面还有一类函数：内联函数，就是使用关键字inline加在定义函数之前，在C里面，调用函数的时候是先把程序执行的当前信息保存到一块特定的内存区域，然后为函数的执行申请内存，函数执行完毕后释放函数的内存空间，返回函数执行信息，接着下面的程序继续执行，这样明显是耗费了一定时间和内存空间保存信息什么的，但是内联函数就省去了这一项，就是不把它当C语言里面的函数处理，不需要保存函数信息什么的，但是内联函数不能含有循环、分支等语句，也不能进行异常处理。（我试了，这些在编译器里面编译都能通过，但是好像就不把它当内联函数处理了，赵聪学长说的）看下面例子：


#include<iostream>


using namespace std;


inline void swap(int &a,int &b)


{

a=b;


b=c;

{

cout<<a<<”   “<<b<<endl;

cout<<a<<”   “<<b<<endl;

C++里面还有一个很重要的东西：函数的重载。


在C语言里面，函数的定义是不能重名的，一旦重名编译的时候编译器报错，但是在C++里面却不同了，C++里面的函数能实现重载，你可以把功能相同的函数取同一个名字，然后调用的时候系统根据函数变量进行区分，如下：


#include<iostream>


using namespace std;


void swap(int &a,int &b)


{


int c;


c=a;


a=b;


b=c;


}


void swap(float &a,float &b)


{


float c;


c=a;


a=b;


b=c;


}


int main()


{


int a=2,b=3;


float c=1.1,d=2.2;


swap(a,b);


cout<<a<<”   “<<b<<endl;


swap(c,d);


cout<<c<<”   “<<d<<endl;


return 0;


}

还有一类函数缺省参数的，就是在定义函数的时候给形参一个默认值，在调用函数的时候有时候可以不用传进去参数，这时候就使用定义函数的时候给的默认参数值。比如定义函数的时候可以这样：


int add(int a=1,int b=2)


{


return a+b;


}


调用的时候可以这样调用：


int a=add();
//a的值为1+2=3；


int a=add(5);

//a的值为5+2=7；


int a=add(2,3)
//a的值为2+3=5；


当然，还可以这样定义：


int add(int a,int b=1)


{


return a+b;


}


这时候记住，有默认值的一定在后面，想想也知道，调用的时候只传一个变量的话肯定先把值赋给前面的，如果定义在后面的参数没有默认值的话那后面的就传不进来参数，不就乱了吗。但是在调用的时候一定要给没有默认值的参数传值！！！看下面例子：

下面讲面向对象：


在C++里面最重要的就是面向对象。也就是类的定义和使用，也就是把所有有统一特性的对象集合起来组成一个类，在C语言里面我们可以使用结构体来定义相同属性的变量，比如我想定义一个点，我们就用


struct point


{


float x;


float y;


};


想定义线段我们就用


struct line


{


float x1;


float x2;


float y1;


float y2;


};


但是在C++里面一般不这么用，在C++里面使用class关键字定义“类”


比如:


class person


{


string name;


int age;


};


在类里面可以定义函数、变量等成员。我们把这些对象分成三个访问级别，分别为私有型（private）,C++里面的默认都是私有型，保护型（protected）,和公有型（public），private类型的只有本类的成员能访问，protected暂时不讲，public是任何成员均能访问。

拷贝函数：


就是声明类A，然后用类A做模板声明类B，此时，类B的当前值跟A一样，相当于复制，但是比复制的效率要高！格式为现在类里面声明函数，<类名>（类名&形参名）。例如：


#include<iostream>


using namespace std;


class Point


{


float x,y;


public:


Point(float a=0,float b=0)


{


x=a;


y=b;


}


Point(Point& P);

//声明拷贝函数


void get(float a,float b)


{


x=a;


y=b;


}


void show()


{


cout<<x<<”     “<<y<<endl;


}


};


Point::Point(Point& P)
//定义拷贝函数


{


x=P.x;


y=P.y;


cout<<“Yes!”;


cout<<endl;


}


int main()


{


Point P1(1,1);


P1.get(1.3,2.4);


Point P2(P1);


P1.show();


P2.show();


return 0;


}
