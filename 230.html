在宿舍憋了两天，调了两天程序，终于调试好了，操作系统不好调试啊，一个小问题就得调试N天。。。。。

关于C语言指针的使用，我们知道指针其实就是一个变量，一般是long型的变量，而指针里面存的地址就是这个long型的值，那么在我们的操作系统的保护下你的指针不能随便指，如果你这样用：
unsigned int * pointer=(unsigned int *)0x874ff;
那么在访问的时候一般会出错，在没有操作系统的情况下就可以了，那么这里发现一个问题，也是我这两天一直调试一直纠结的问题就是，指针在外面申请的时候不能初始化，或者说可以初始化为0，0就是NULL的值，在没有头文件的情况下是没有NULL的，就可以直接写0.
那么如果这样用：
unsigned int * pointer =  (unsigned int *)0x00100000;
在以后的函数里面对0x00100000这块地方操作的时候就会出错，那么可以这样使用，就是先申请，再赋值：
unsigned int * pointer = 0;
在后面的函数里面使用的话再赋值指明指向内存哪块地方：
pointer = (unsigned int
*)0x00100000;
这里就是指向内存的1MB处，这样在函数里面就可以操作了。否则就容易出错。
