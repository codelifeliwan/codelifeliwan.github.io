73-暂时禁用一个用户账号

如果你的看过“秘籍50”，里面论述了如何让Ubuntu建立儿童账号，防止他们访问“儿童不宜”的Internet网站。现在，你可以有的时候不希望他们上网，譬如希望他们做一些其他的事情。

 

下面的内容，将帮助你暂时禁用儿童账号：

Sudo password –l username

其中username就是用户名。

 

如果想启用“已经暂时禁用的账号”，只需：

Sudo password –u username

74-完全掌控控制桌面效果

如你所知，Ubuntu包含了几个桌面效果和动画。譬如，当窗口缩小的时候，会出现缩放到面板的效果。你可以从“系统”-》“首选项”-》“外观”里面选择“可视化效果”标签里面的“Extra”按钮，来得到更多的效果。

 

为了对于“制桌面效果”有更多的定制性，你可以安装下面的两个软件包之一。

第一个是“Compiz Config Setting
Manager”，可以在“新立得”里面的simple-ccsm软件包里面找到。安装完成以后，就可以在“系统”-》“首选项”里面找到了。它运行非常简单明了，它可以提高比默认的“桌面效果”更多的选项。它提高几个标签-“Animations”、“Effects”、“Desktop”、“Accessiblity”、“Edges”等几个标签，其中“Animations”可以让你改变“最大化/最小化”窗户的视觉效果，“Effects”可以改变“Alt+Tab”键时的视觉效果，“Desktop”提高虚拟桌面切换时的效果，“Accessiblity”改变屏幕效果，“Edges”提高鼠标移动到屏幕边缘时的效果。

另外一个具有更加可配置性的软件是“Compizconfig-settings-manager”（如果你安装了Simple Compiz
Config Setting
Manager，这个软件就已经安装了）。这个软件可以开启或者屏蔽某些桌面效果插件，并且提高更加细致的数值选项。

 

 

Ubuntu的桌面效果并不是非常容易理解，一个很好的起点是Ubuntu论坛Ubuntuforums.org。

75-进行桌面出版

Ubuntu可以进行桌面出版的设计。Scribus是一个专业级别的DTP软件，完全可以媲美Adobe Indesign和Quark
Xpress。它提高了CMYK颜色、色彩分类等许多功能。现在，几个主流的出版社都使用它进行工作。安装完成Scribus以后，可以在“程序”-》“办公软件”里面找到它。

 

需要Adobe Illustrator或者Core
Draw那样的矢量绘图功能吗？没有问题，你可以在“新立得”里面安装inkscape软件包。Inkscape是一个专业级别的矢量绘图软件，可以完成任何绘图工作。

实际上，GNOME的许多artwork就是它完成的。它包含矢量点编辑、复杂的路径操作、跟踪图形等功能。Inkscape的文件输出是工业标准格式SVG。你可以在“程序”-》“图形”里面找到它。

 

如果你希望创造自己的TrueType字体，或者修改已有的TrueType字体，可以安装Fontforge软件。

77-搜索Ubuntu的文件

人们总是花费一些时间来寻找丢失的东西，而Ubuntu至少在计算机方面提供了一些帮助，提高了非常强劲的搜索界面，包含命令行和GUI。

 


命令行搜索

 

有两个命令行工具可以搜索文件：locate和find。他们的不同之处在于，locate建立在一个文件数据库上面，而find则每一次都重新搜索要寻找的文件。

 

locate基于一个后台程序-updatedb。updatedb在系统后台运行，每一次有系统更新updatedb数据库。locate的搜索结果有一个明显的缺点就是，它依赖于系统的最近更新时间。所以，一般在运行locate以前，命令行运行updatedb（需要root权限）。locate有好几个不同版本，Ubuntu携带的locate（名字叫做“mlocate”），只能找到新增加的文件，因此不必每次都先运行updatedb。

 

使用locate非常简单，只需输入“locate
需要搜索的内容”就可以了。譬如搜索文件名带有“fstab”的文件，只需需要“locate
fstab”就可以了。locate还支持通配符，譬如搜索所有的MP3文件，可以输入“locate *.mp3″。

 

使用find则有所不同。首先你必须输入搜索的位置和搜索内容。譬如搜索你的HOME目录里面的文件accounts，必须输入”find
/home/yourname -name accounts”.

 

locate和find都支持正则表达式。正则表达式的具体内容超出了本身的范畴，不过在google可以找到几本入门几本的书籍。正则表达式是所有使用Linux的人们必须花费时间学习的一个工具。

 


桌面搜索

 

Ubuntu桌面已经包含了几个Tracker工具。像Vista和Mac OS X里面的工具一样，这些工具可以搜索所有的数据，而不仅仅是
文件。它可以查看你最近查看的网站，还会索引Email的内容和附件，甚至可以搜索PDF文件里面的内容。

Tracker可以变成Ubuntu的一个系统访问点。你不必打开Nautilus，然后慢慢找到需要的文件。你只需要打开搜索窗口，输入它的名字，文件就出现在搜索结果里面，双击打开就可以了。如果你想打开OpenOffice
Writer，不需要从“程序”-》“办公软件”-》“OpenOffice.org
Writer”，只需打开搜索窗口，输入Writer，双击搜索结果里面的Writer程序就可以了。

 

Tracker在Ubuntu
8.04里面并没有被启用，因为有人认为它降低了系统的性能。因为它在后台自动运行，跟踪系统的运行以及他们需要的资源，还在每一次你登录的时候，会进行一个索引操作。也就是说，尽管有很多理由不运行它，但是还是值得一试。下面，将讲述如何开始和关闭Tracker。

 

在开启Tracker之前，最好安装一下ww软件包-可以索引Word文档里面的内容。现在打开”系统”-》“首选项”-》“搜索和索引”，勾选“运行索引和监控”框。如果使用notebook，你可以还有勾选“使用电池时不要使用索引”。

 

点击OK按钮，会提示“Tracker服务已经重新启动”，一个沙漏图标会出现的面板上面。但是现在还不能使用Tracker工具，因为在使用Tracker以前，你必须让Tracker索引一下你的文件。索引文件可能需要花费一段时间。如果你看到Tracker图标变成了一个黄色的沙漏，这表示Tracker已经完成了索引工作。下一次，如果图标发生改变（时间会很短），就表示Tracker在跟踪文件的改变。

 

使用Tracker，使用点击面板上的图标，输入搜索内容，然后点击“Find”按钮就可以了。搜索结果的类型在左边，具体搜索结果在右边，双击即可打开。

 

Tracker还可以在命令行使用（需要在新立得里面安装tracker-utilus软件包），只要输入“tracker-search
内容”就可以了。

78-移除“密码错误”等待时间

在Ubuntu里面，如果输入了不匹配的密码，Ubuntu会禁止你在两分钟以内无法登录。这是为了防止黑客用”暴力破解“的方法得到你的密码。一台计算机现在可以在一分钟以内尝试成百上千此的密码重试，两分钟的延迟提供了一种保护措施。

 

但是，如果你-像我一样，总是打错密码，你可能希望把这个禁止登录时间改为0，即没有禁止登录时间。

 

用gedit打开/etc/pam.d/common-auth文件：

gksu gedit /etc/pam.d/common-auth

 

找到：

auth   
requisite      
pam_unix.so nullok_secure

把它改为：

auth   
requisite      
pam_unix.so nullok_secure nodelay

 

现在保持退出，重新启动。

 

现在可以测试一下了，故意输入一个错误密码，看看发生了什么。

 

这个做法会降低系统的安全强度，慎用。

有的时候看不清图标？只需右键点击桌面图标，选择“伸展图标”，在图标周围会出现拖动把手。几乎所有的图标可以拖动到屏幕大小，但是如果是SVG格式的图标，看起来会更加美观。系统缺省的Human主题的图标都是SVG格式。如果想变小，只需右键选择“恢复图标原始大小”就可以了。

 

80-运行Ubuntu但是不是在Linux上面

如果你不喜欢Linux内核，你可以使用Nextenta。它使用Sun公司的OpenSolaris作为系统核心，而不是使用Linux。OpenSolaris不像Linux，有着更纯正的Unix血统。

 

OpenSolaris内核可能更适合于服务器，同时硬件驱动也相对较少。但是很多人OpenSolaris是开源世界的明日之星。OpenSolaris包含了dtrace等一些强有力的系统工具，相对于Linux提供了一个更高级的功能。

81-瞬间隐藏文件和文件夹

在Nautilus，如果一个文件或者文件夹的名称以”.”开头，则被认为是隐藏的，会不显示在一些shell命令的结果，譬如ls，除非使用ls
-a命令。如果想隐藏一个文件或者文件夹，只需键入”F2“，然后在文件名的前面，加上一个”.”就可以了。如果还没有隐藏，按F5刷新一下即可。

 

如果你希望在Nautilus里面隐藏文件，但是在shell命令里面可以看到这些文件，加入一个”~”在文件名的末尾，譬如想隐藏“partypicture.jpg”，只需文件名改为”partypicture.jpg~”就可以了。

82-病毒扫描（现在一般用avast）

一般来说，Ubuntu不需要对于病毒的事情担心，可能针对Linux的病毒数量不会超过50个。通常这些病毒影响服务器软件（譬如Apache），作为桌面用户，Linux完全不必担心。

 

尽管如此，安装一个防病毒软件还是有备无患。

 

下面将讲述如何安装ClamTK。ClamTK是一个ClamAV的GUI程序。ClamAV是一个重负载主机下面工作的工具，但是没有原因不可以把它用到你的桌面电脑上面。

 


安装配置ClamTK

 

在新立得里面，搜索“clamtk”软件包，安装它。安装完成以后，可以在“程序”-》“系统工具”里面找到它。

 

 

在使用ClamTK扫描病毒以前，第一次运行必须首先更新一些病毒库（需要root权限）。打开一个终端，输入

sudo clamtk

 

选择菜单“帮助”-》“更新。。”。

更新完成以后，关闭ClamTK。现在从“程序”菜单里面打开。注意，ClamTK下一次会自动在后台更新病毒库了，不必每次都这样手工更新。

 

 


扫描病毒

 

扫描整个系统，点击“Options”菜单，选择“Scan Hidden Files”。现在点击“File”-》“Rescursive
Scan”，从对话框里面选择“file System”，选择OK!

注意的几点：

  
1、全盘扫描，是一个高CPU使用率的工作。因此，在扫描期间，ClamTK会出现假死现象。

  
2、在Ubuntu里面，有几个文件夹ClamTK是不会扫描的（譬如/proc），ClamTK会报告”Exluded”。

   3、如果出现“false
positive”的提示，表示ClamTK认为一个文件包含病毒，但是无法确认病毒的名称。你可以用“文件名+clamav”的方式，在Google里面找到更具体的解释。

 

        
如果你觉得时间太长，可以让ClamTK只扫描你的home目录。HOME通常放置这你下载的文件，而下载的文件通常是病毒最容易藏身的地方。

 

        
如果发现了病毒，ClamTK并不会从文件里面删除这些病毒，替代的是，决定权在于你。当然大部分病毒都是一些Windows病毒，在Linux对你毫无危害。

 

        
ClamTK包含一个“隔离”功能，它把病毒文件拷贝到一个特殊的文件夹，你可以依然用Nautilus浏览这个文件夹，删除里面的文件或者进行进一步的处理。

 

添加一个“scan-on-demand”功能

你可以建立一个Nautilus右键“杀毒”菜单。

打开Gedit，在HOME目录建立一个virus_scan文件，内容如下：

#!/bin/bash

# Scan the selected file in clamtk

clamtk $@

保存退出。

更改权限，并拷贝到Nautilus的右键script区，打开一个终端：

chmod +x virus_san

mv virus_san .gnome2/nautilus-scripts/

 

现在在文件或者文件夹上面点击右键，选择“Script”->”virus_san”就可以了。

如果你必须立刻离开你的终端窗口，你可能希望立刻启动屏幕保护，同时如果设置了屏幕保护密码，还可以保护系统。
