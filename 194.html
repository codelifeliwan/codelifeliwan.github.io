JSP语法基础


注释：

JSP注释有两种，一种是程序员注释，一种相当于用户注释，程序员注释在网页的查看源代码中不可见，而用户注释是可见的， 在用户注释中可以加入自己的代码：

 

用户注释：<!– 注释内容–>

程序员注释：<%–注释内容–%>

例如在用户注释中显示时间：<!–现在时间为：<%=new java.util.Date().toLocalString()%>–>

 


声明变量和函数：

JSP中声明变量和函数使用下列格式：

<%!声明部分%>

例如：

<%!int i=1;%>

<%!

public String getIp(){

return request.getRemoteAddr();

}

%>

 


使用JSP表达式输出结果：

输出结果格式为：

<%=表达式%>

注意这里的表达式必须有返回值例如：

<%=getIp()%>

 


使用JSP脚本实现程序逻辑：

格式为：

<%程序%>

例如：

<%for(int =0;i<10;i++){%>

<%number++;%>

<%}%>

 


page指令：

用法：

<%@ page…%>

一般用来指定MIME类型、字符编码方式和引入包等。

引入包：

<%@ page import=”包名“%>

指定MIME类型：

<%@ page contentType=”类型“%>例如：

<%@ page contentType=”text/html”%>

指定字符编码：

示例方法一、<%@ page content type=”text/html:charset=GBK”%>

示例方法二、<%@ page pageEncoding=”UTF-8″%>

 


包含


静态包含：

静态包含使用<%@ include file=”PAGE”%>

静态包含是原封不动的插入进来，就像是C语言的宏定义一样。例如：

<%@ include file=”test.jsp”%>

就是把test.jsp的内容原封不动的插入此处。

 


动态包含：

动态包含是使用<jsp:include page=”PAGE” …  />

动态插入的可以动态的传递参数，这事静态包含所不具有的。

两种使用方法：

·1、<jsp:include page=”PAGE”  …   />：将在目标页面输出PAGE页面输出的内容。

·2、<jsp:include page=”PAGE” …>

<jsp:param name=”name” value=”值” />

…

 </jsp:include>

这样包含可以取得前一页面的值。

 


跳转：

<jsp:forward page=”PAGE”  />

PAGE的值可以动态计算。

<jsp:forward page=”<%=表达式%>”>

 

Forward还有一种用法和jsp:include一样。可以不同页面之间传参数。

 

 


用标签操作JavaBean：

JavaBean不多说，在jsp里面两个字：重要！

那么JavaBean的使用方法如下：

先声明一个类（一般需要打包）例如：

package wanli;

 

public class Test001 {

 private String name=”wwwl”;

 public Test001(){

 name=”Hello Default”;

 }

 

public String getName() {

return name;

}

 

public void setName(String name) {

this.name = name;

}

}

注意：这里报名类名什么的不能随便乱取，不能跟系统冲突，否则会错误，此类错误特别多！！！

类里面一般会有一个共有无参构造方法，用来实例化，属性有get和set方法，一般会将属性名首字母小写，get和set方法色属性名则大写。

编译好的.class文件放在WEB-INF/classes里面。

 

引用一般会使用如下标签：

<jsp:useBean id=”name” classes=”类全名”  />

<jsp:setProperty name=”name” property=”属性名” value=”属性值”  />

<jsp:getProperty name=”name” property=”属性名”  />

JavaBean是通过get和set方法来设置和取得属性的，这里，name是用户定义的标识，名字不要取太特殊，否则冲突了容易出错。Property是属性名。是类里面需要set和get的属性的名称。Value是想要设置的属性值。

如下例子：


<body>


<jsp:useBean id=“aaa” class=“Demo.Test.Wan” />


<jsp:setProperty name=“aaa” property=“name” value=“Hello LiWan” />


<p><jsp:getProperty name=“aaa” property=“name” />


</body>

 

页面就输出Hello LiWan
