JSP内置对象–session

session对象表示javax.servlet.http.HttpSession对象。

session对象的表示范围是整个浏览器范围，在浏览器不关闭的情况下有效。


session对象的主要方法：

Public void setAttribute(String name,Object obj)：设置属性。

Public Object getAttribute(String name)：取得属性。

Public void removaAttribute(String name)：删除属性。

Public boolean isNew()：判断是否是新的session。

Public String getId()：返回session。

Public void invalidate()：让session失效。
