UVA 465 – Overflow解题报告【大数加或者乘】

Posted on 15 五月, 2011 by wanli — No Comments ↓

题目意思很简单，就是两个大数运算，加或者乘，如果超出int型范围，就输出。

首先是套了zcoder的模板，没过。。。。。好像是前导0的问题，这题关键是把前导0去掉，后来自己重新写了一遍，出了一些小问题，然后AC了。
此题可以在里面多加几个判断，可能会优化点。
原题：

Overflow 

Write a program that reads an expression consisting of two
non-negative integer and an operator. Determine if either integer
or the result of the expression is too large to be represented as a
“normal” signed integer
(type integer if you are
working Pascal,
type int if you are
working in C).

 

Input

An unspecified number of lines. Each line will contain an integer,
one of the two
operators + or *,
and another integer.

 

Output

For each line of input, print the input followed by 0-3 lines
containing as many of these three messages as are appropriate:
“first number too big”, “second number too
big”, “result too big”.

 

Sample
Input

 

300 + 3 

9999999999999999999999 + 11

 

Sample
Output

 

300 + 3 

9999999999999999999999 + 11 

first number too big 

result too big

{

 for(i=0;i<len;i++)

 {

 if(k==0&&s[i]!='0')

 {

 k=1;

 }

 if(1==k)

 {

 tmp[j++]=s[i];

 }

 }

 if(j==0)j++;

 tmp[j]='';

 memcpy(s,tmp,sizeof(tmp));

}

{

 int len1=strlen(n1);

 int len2=strlen(n2);

 int len11=strlen(re);

 for(int i=0;i<len11;i++)re[i]=0;

 int tmp=0;

 int i=0,j=0;

 for(i=max(len1,len2)-1,j=0;i>=0;i--,len1--,len2--,j++)

 {

 if(len1>=1)re[j]+=(n1[len1-1]-'0');

 if(len2>=1)re[j]+=(n2[len2-1]-'0');

 re[j]+=tmp;

 tmp=re[j]/10;

void one_mul(char *n,int a,char *re) //大数和a相乘，结果存在re里面

{

 if(0==a){re[0]='0';re[1]='';return;}

 int len=strlen(n);

 int j=0,tmp=0;

 for(;len>=1;len--,j++)

 {

 re[j]=(n[len-1]-'0')*a+tmp;

 tmp=re[j]/10;

 re[j]=re[j]+'0';

 }

 if(tmp!=0)re[j++]=tmp+'0';

 re[j]='';

 char c1;

 for(int i=0;i<(j/2);i++)

 {

 c1=re[i];

 re[i]=re[j-i-1];

 re[j-i-1]=c1;

 }

}




void mul(char *n1,char *n2,char *re) //两个大数相乘，结果存在re里面

{

 char tmp[maxn];

 char tmp1[maxn];

 int len1=strlen(re);

// memset(re,0,sizeof(re));

 {

 one_mul(n1,n2[i]-'0',tmp);

 int k1=j+(len2-i)-1;

 tmp[j]='0';

 tmp[j]='';

 memcpy(tmp1,re,sizeof(tmp1));

// memset(re,0,sizeof(re));

 add(tmp1,tmp,re);

 }

}




int mycmp(char *n1,char *n2) //比较两个数大小

{

 if(strlen(n1)==strlen(n2))return (strcmp(n1,n2));

 else

 {

 return (strlen(n1)>strlen(n2)? 1:-1);

 }

}
