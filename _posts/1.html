C语言指针使用

Posted on 26 十一月, 2010 by wanli — 1 Comment ↓




调试了一天的程序，总是在指针建立二叉树的那个地方停住了，现在总算明白了，为了加深印象，写此日志。

首先，指针是指向地址的这个明白，为了说明几个常见错误，还是用代码说说吧。。。。。
看下面代码：
#include<stdio.h>
#include<memory.h>
int main()
{
char c[10],*c1=c;
memset(c,0,10);
c1=”hello”;
printf(“%sn%sn”,c,c1);
return 0;
}
这个是我以前犯的错误。。。。。输出的时候第一个什么也不输出，第二个输出字符串。因为首先是申请指针把c1指向c，可是后来执行 c1=”hello”;以后c1不指向c了，相当于在内存中重新开辟了一段内存空间，c1指向了重新开辟的那段内存空间了而不是给c赋值。。。。。
再来看看下面一段代码：

#include<stdio.h>

#include<stdlib.h>




void chuli(int *a)

{

if(NULL==(a=(int *)malloc(sizeof(int))))

{

printf(“can not use memory!n”);

exit(0);

}

*a=100;

}




int main()

{

int *a=NULL;

chuli(a);

if(NULL==a)

printf(“NO!n”);

else

printf(“YESn%dn”,*a);

return 0;

}

执行结果是NO！为什么呢？？？有的同学会说我传进去函数的是指针啊，为什么还不变呢？？？

这个要关系到存放指针的地址问题，在chuli（）函数里面给指针重新分配了内存，然而传进函数里面的地址是不变的，也就是说在函数里面变量a指向了另外一个新开辟的内存空间，然而老的内存空间是不会变的，传进函数里面的是a所指向的地址，而不是a，a可以说是个临时变量，函数里面只是把新开辟的内存空间赋给临时变量。一句话，在这个函数里面要改变main()函数里面a的地址是不可能的，就是说在main()函数里面一旦一个变量被申请，那么存放它的地址是固定的，不会再改变，那么如何解决这个问题呢？我只要往上一级，使用指向地址的指针，也就是指向指针的指针，然后，指向指针的指针所指向的地址是可以改变的，而不需要本指针的地，看下面修改过的代码：

#include<stdio.h>

#include<stdlib.h>




void chuli(int **a)

{

if(NULL==((*a)=(int *)malloc(sizeof(int))))

{

printf(“can not use memory!n”);

exit(0);

}

**a=100;

}




int main()

{

int *a=NULL;

chuli(&a);

if(NULL==a)

printf(“NO!n”);

else

printf(“YESn%dn”,*a);

return 0;

}

这样，输出的是YES   100了。。。。。

当然，这个在C++里面使用引用更简单，而在C里面只能使用这个，或者可以使用全局变量。

比如下面一个建立简单二叉树的代码（不正确的）：

#include<stdio.h>

#include<stdlib.h>







struct tree

{

int data;

struct tree *lchild;

struct tree *rchild;

};




void BuildTree(struct tree *S) //构造二叉树

{

int data;

scanf(“%d”,&data);

if(0==data)

{

S=NULL;

}

else

{

if(NULL==(S=(struct tree *)malloc(sizeof(struct
tree))))

exit(0);

S->data=data;

BuildTree(&(S->lchild));

BuildTree(&(S->rchild));

}

}




void Print(int a)

{

printf(“%d   “,a);

}




void PreView(struct tree *S) //先序遍历二叉树

{

if(S)

{

Print(S->data);

PreView(S->lchild);

PreView(S->rchild);

}

}




void MidView(struct tree *S) //中序遍历二叉树

{

if(S)

{

MidView(S->lchild);

Print(S->data);

MidView(S->rchild);

}

}




void BackView(struct tree *S) //后续遍历二叉树

{

if(S)

{

BackView(S->lchild);

BackView(S->rchild);

Print(S->data);

}

}




int main()

{

struct tree *S;


//
freopen(“test.txt”,”r”,stdin);

BuildTree(S);

PreView(S);

printf(“n”);

MidView(S);

printf(“n”);

BackView(S);

printf(“n”);

return 0;

}

这个跟上面一样，在二叉树建立的时候传进函数指针，然而编译没有错误，运行就不对了，函数并不能改变存放S的地址，也不能返回其地址，所以修改如下：

#include<stdio.h>

#include<stdlib.h>







struct tree

{

int data;

struct tree *lchild;

struct tree *rchild;

};




void BuildTree(struct tree **S) //构造二叉树

{

int data;

scanf(“%d”,&data);

if(0==data)

{

(*S)=NULL;

}

else

{

if(NULL==((*S)=(struct tree *)malloc(sizeof(struct
tree))))

exit(0);

(*S)->data=data;

BuildTree(&((*S)->lchild));

BuildTree(&((*S)->rchild));

}

}




void Print(int a)

{

printf(“%d   “,a);

}




void PreView(struct tree *S) //先序遍历二叉树

{

if(S)

{

Print(S->data);

PreView(S->lchild);

PreView(S->rchild);

}

}




void MidView(struct tree *S) //中序遍历二叉树

{

if(S)

{

MidView(S->lchild);

Print(S->data);

MidView(S->rchild);

}

}




void BackView(struct tree *S) //后续遍历二叉树

{

if(S)

{

BackView(S->lchild);

BackView(S->rchild);

Print(S->data);

}

}




int main()

{

struct tree *S;

freopen(“test.txt”,”r”,stdin);

BuildTree(&S);

PreView(S);

printf(“n”);

MidView(S);

printf(“n”);

BackView(S);

printf(“n”);

return 0;

}

这样就OK了！
