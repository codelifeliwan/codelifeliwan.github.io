86—命令行查看图片


在命令行里面查看工具，可以输入eog
filename。例如，你想查看holiday.jpg，可以输入eog
holiday.jpg。holiday.jpg就会显示了，如果想全屏幕显示，输入eog
-f holiday.jpg。

87-通过Web浏览器管理打印机

同其他的Linux系统一样（其实Macintosh OS X也一样），Ubuntu使用CUPS作为打印系统。

 

你可以在浏览器里面输入“localhost:631”，打开CUPS控制面板。在打印机标签里面可以配置所有的打印机，在Jobs标签里面可以查看所有的打印作业（可能需要刷新）。

 

88-不必点击标题栏就可以移动窗口的方法

前提是窗口没有被最大化。按下Alt，然后点击拖放窗口的任意部位，鼠标会自动变成一个小手，现在就可以自由移动窗口。

 

这个方法，适用于窗口的标题栏在屏幕以外，或者因为显示器太小，而无法看到全部窗口的情况。

89-使用命令行连接“共享文件夹

如果在办公室或者家里，你拥有多台电脑，你可能需要通过“共享文件夹”来互相访问。Ubuntu的位置-》网络可以让你看到网络内的其他电脑。

 

但是如果想通过命令行来访问“共享文件夹”，还是有一点点小小的问题。一旦在Nautilus里面访问了共享文件夹，这个共享文件就会被挂载到你的HOME目录下面的隐藏文件夹.gvfs下面。如果桌面没有开启或者共享文件夹没有挂载，就会无法访问。

 

在这种情况下，你可以使用“smbclient”。smbclient提供了一种类似于ftp方式的共享文件访问方式，并且命令也非常类似于ftp。详细介绍：

1、smbclient的-L参数，列出网络计算机上面的的共享，参数可以是计算机名称，也可以是计算机的IP地址。譬如查看一台名字叫keir-windows的计算机上面的共享，可以使用：

smbclient -L keir-windows

可能需要输入一个密码！按Enter会显示所有的共享资源。

2、连接到共享文件夹。和Windows类似，smbclient使用“”来表示地址。但是因为““是Linux的一个命令行特殊字符，因此必须告诉Linux“”真正含义。譬如连接到资源\keir-windowssharefolder，就变成了\\keir-windows\sharefolder:

smbclient \\keir-windows\sharefolder\

 

90-取消CapsLock键的功能

如果你无意间按到“Caps Lock”键，所有键入的字母就变成了大写。如果关闭这个功能呢？

 

打开一个终端，输入：

xmodmap -e ‘clear lock’

就可以禁用‘Caps lock“键了。现在即使”Caps Lock“的指示灯亮着，对于大小写输入也没有任何影响。

 

要想一登录就禁用“Capslock”键，可以把这一行添加到HOME目录的.profile文件里面的后面，重新登录使设置生效。

 

91-格式化软盘

还在使用软盘？没有问题。按下Alt+F2，输入“gfloppy”就可以了，一切和Windows里面的操作类似。

 

92-使用轻型的文件管理器

Thunar是Xubuntu使用的文件管理器。它启动快速，内存占有很少，但是功能却非常强劲，能够提供你所需要的全部特性。在某些特性方面，Thunar明显比Nautilus强大一些。因此它可以替代Nautilus，但是还可以保留Nautilus，以使用Nautilus的某些特性，譬如CD-R/DVD
Creator.

 

现在开始安装：

1、打开新立得，找到并安装“thunar”和“thunar-archive-plugin”软件包。安装完成以后，可以在终端里面输入thunar就可以启动thunar了。

 

2、改变设置，使用thundar打开文件夹。键入：

gksu gedit
/usr/share/application/nautilus-folder-handler.desktop

 

把这一行：

Exec=nautilus –no-desktop %U

改为：

Exec=thunar %U

 

保持退出。现在在菜单“位置”-》“主文件夹”测试一下。

 

其中，上面的方法，也适用在其他的文件管理器上面。如果你喜欢Konqueror（KDE的文件管理器）、Dolphin（KDE4的文件管理器）、Rox-filer（另外一个轻型的文件管理器），你也可以这样安装设置一些就可以了。不过改变设置的时候，Rox-filer不需要%U参数，但是Konqueror和Dolphin需要。

 

如果你喜欢命令行文件管理器，可以安装Midnight
Commander（在新立得里面搜索“mc”软件包安装）。安装完成以后，在终端里面输入“mc”启动。如果你使用过Dos里面运行的Norton
Commander软件，你会发现操作非常熟悉。

 

93-使用Gedit的“关键词高亮显示”功能

程序员可能对于Gedit的“关键词高亮显示”功能非常感兴趣。但是在你保存一个文档以前，这个功能是不会被触发的。

gedit还提供一个”匹配括号“的功能，可以在”编辑“-》”首选项”里面，勾选“突出显示匹配的括号”就可以了。

 

如果想禁用gedit的高亮显示，可以打开gconfi-editor，找到/apps/gedit-2/preferences/synatx
highlighting，不勾选“Enable”就可以了。

 

94-防止Email附件里面的Zip文件被误报病毒

如果你使用“创建归档文件”功能来建立一个Zip文件的时候，默认的Zip文件名称是原来的文件名+”.zip”.譬如你压缩一个文件document.doc，得到的zip文件名是document.doc.zip。现在的问题是，有些Windows防病毒软件认为如果一个文件名的扩展名包含两个，可能是病毒。为了避免这种情况，请去掉第一个扩展名，即改名为document.zip.

 

95-使用另外的Email客户端软件

Ubuntu默认的Email客户端软件是Evolution。它现在提高的功能越来越多，越来越面向与商业用户。Evolution现在非常类似于Outllok，提供了一大堆我感觉并不需要的功能。现在，我介绍两个Email客户端软件，他们都可以从“新立得”里面轻松安装，并且都非常简单易用。

 

Claws Mail

虽然这个软件比较小巧，但是功能也不弱，并且和GNOME桌面集成的非常好。现在的Email客户端好像更喜欢安排我们的生活，提供“日历”和“to-do-list”等功能。Claws
Mail并没有提供这些，甚至于撰写HTML格式的邮件格式化也不支持（但是能看），就是一个专门处理Email的软件。它支持“拼写检查”和“邮件过滤”功能，但是如果你希望得到其他功能，也可以到它的网站下面下载插件。

 

你可以在新立得里面搜索”claws-mail”，就可以安装了。安装完成以后，可以在“程序”-》“互联网”里面找到它。

 

Thunderbird

正如一些人知道的那样，Firefox并不是Mozilla的唯一产品，Thunderbird就是Mozilla提供的email客户端软件，并且也是是当今第二大开源Email客户端软件。Thunderbird的功能和人们预期的一样强大，界面美观、搜索强劲、可以进行垃圾邮件检查，但是它依然保持着“简单易用”的特性。Thunderbird还可以阅读Usenet新闻组，可以收发Gmail邮件。Thunderbird的Html渲染，与Firefox使用同样的技术。

 

同Firefox一样，Thunderbird也可以通过“附加功能“进行扩展。你可以从Thunderbird的网站下载需要的插件。

 

安装Thunderbird，最后的方式是在新立得里面安装”thunderbird-gnome-support“包。这个软件包可以保证与GNOME最好的集成性。如果你希望安装本地语言界面，可以搜索”thunderbird-locale“来安装本地语言包。

 

安装完成以后，可以在“程序”-》“互联网”里面找到它。首次运行，会出现一个向导，让你配置邮件账号。

 

96-确保当你使用话筒的时候其他人可以听到你的声音

首先必须确保打开话筒，在GNOME面板是的“音量控制”图标双击，会出现窗口，保证“话筒”不是被关闭：

点击话筒下面的红叉，打开话筒。

如果还不够大，可以打开“Mic boost”功能（声音提高20分贝）。单击“编辑”-》“首选项”，勾选“Mic boost”

希望在“文件系统”里面找东西，可以打开“位置”菜单，键入“/”输入就可以了

 


98-关闭系统Beep

 


如果你出现操作失误，或者Ubuntu希望提示你有些功能无法完成的时候，它会让计算机Beep一下。这可以影响其他人的，所以你可能希望关闭这个功能。打开”系统“-》”首选项“-》”音效“，找到”系统响铃“标签，不勾选”启用系统响铃“功能就可以了。


或者，你希望可以使用”系统视觉响铃“，当有错误发生的时候，可以闪烁整个屏幕或者窗口的标题栏来提醒你。

99-添加第二个硬盘

如果你的硬盘空间非常紧张了，你可以添加新的硬盘（这里指的是没有分区的新硬盘，已经分区的硬盘Ubuntu可以自动找到)。添加新硬盘，必须做两件事情：分区和格式化，另外如果你希望Windows也可以访问新硬盘，你必须使用FAT32格式化。

 

在开始以前，你必须首先查看一下，Ubuntu是否找到了新的硬盘。步骤：

 

1、启动Ubuntu。开启一个终端，输入：

sudo fdisk -l

下面是我的输出结果：

 

Disk /dev/sda: 81.9 GB, 81964302336 bytes

255 heads, 63 sectors/track, 9964 cylinders

Units = cylinders of 16065 * 512 = 8225280 bytes

Disk identifier: 0x1c381c37

Device Boot Start End Blocks Id System

/dev/sda1 * 1 4742 38090083+ 7 HPFS/NTFS

/dev/sda2 4743 9964 41945715 5 Extended

/dev/sda5 4743 9744 40178533+ 83 Linux

/dev/sda6 9745 9964 1767118+ 82 Linux swap/Solaris

 

Disk /dev/sdb: 120.0 GB, 120034123776 bytes

255 heads, 63 sectors/track, 14593 cylinders

Units = cylinders of 16065 * 512 = 8225280 bytes

Disk identifier: 0xb94838a4

Disk /dev/sdb doesn’t contain a valid partition table

 

可以看到，Ubuntu已经找到两块硬盘（/dev/sda和/dev/sdb），并且提高了详细的信息。

 

2、现在使用cfdisk来给新硬盘分区：

sudo cfdisk -z /dev/sdb

现在输入”n“建立一个新分区，回车确认提示信息。输入”t“，再按回车，输入0C（注意是数字0然后加字母C），选择分区类型。输入”w”,写入分区信息。

然后在终端里面，输入下面的命令，可以建立一个FAT格式的分区（便于在Windows下面使用）：

sudo mkfs.vfat -F 32 -n NewDisk -I /dev/sdb1

 

 

其中“NewDisk”是U盘的标签，可以自己改变，/dev/sdb1就是前面的看到的新硬盘设备的Ubuntu设备编号。格式化完毕，一个新的硬盘就会被自动加载到系统了。

 

 

100-Ubuntu后台自动更新

 

Ubuntu会经常更新，让你的系统可以安装最新的安全补丁文件。但是”更新管理器“似乎不断的打扰你，提示”有新的更新下载“。如果你感到厌倦，可以让Ubuntu自动在后台更新。

 

打开”系统“-》”管理“-》”软件源“，点击”更新“标签”，在“自动更新”里面勾选”不确认就安装安全补丁”就可以了。
