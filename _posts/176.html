Java数据库操作：


驱动配置：

首先到www.mysql.com上面下载数据库Java的驱动，在connect/J里面。解压后将里面的.jar文件复制到一个目录下，本次以C盘跟目录为例。

然后再配置classpath，在classpath里面新建，添加一个：.;C:mysql-connector-java-5.1.17-bin.jar即可。

敲入以下代码检验是否正确：

public class Database001

{

public static final String S=”org.gjt.mm.mysql.Driver”;

public static void main(String args[])

{

try{

Class.forName(S);

}catch(Exception e){

e.printStackTrace();

}

}

}

如果程序什么都没输出则配置正确，否则重新配置。


数据库连接及关闭：

使用DriverManager类进行数据库连接操作。

1、public static Connection getConnection(String url)throws SQLException

2、public static Connection getConnection(String url,String user,String password)throws SQLException

其中url为连接地址，user为用户名，password为密码。

url一般格式为jdbc:mysql://IP地址:端口号/数据库名称。

连接上以后就可以进行操作了。

现在只记住数据库关闭操作方法：void close()throws SQLException

看下面代码：

import java.sql.Connection;

import java.sql.DriverManager;

import java.sql.SQLException;

import javax.swing.*;

public class Databaseconnection01

{

//
public static final String S=”org.gjt.mm.mysql.Driver”;

public static void main(String args[]){

Connection conn=null;

 

String DBURL=”jdbc:mysql://localhost:3306/testdata”;

String DBUSER=JOptionPane.showInputDialog(“Please input username:”);

String DBPASSWORD=JOptionPane.showInputDialog(“Please input password:”);

try{

conn=DriverManager.getConnection(DBURL,DBUSER,DBPASSWORD);

JOptionPane.showMessageDialog(null,conn+”n数据库连接正常！“, “”,JOptionPane.PLAIN_MESSAGE);

}catch(SQLException e){

e.printStackTrace();

}

try{

conn.close();

JOptionPane.showMessageDialog(null,”数据库关闭正常！“, “”,JOptionPane.PLAIN_MESSAGE);

}catch(SQLException e){

e.printStackTrace();

}

}

}

程序从键盘输入用户名密码连接数据库并且关闭数据库。

数据库的插入、删除、修改：

看下面代码：

import java.sql.Connection;

import java.sql.DriverManager;

import java.sql.Statement;

import java.sql.SQLException;

import javax.swing.*;

public class Databaseconnection01

{

public static final String DBURL=”jdbc:mysql://localhost:3306/testdata”;

public static final String S=”org.gjt.mm.mysql.Driver”;

public static void main(String args[])throws Exception{

Connection conn=null;

Class.forName(S);

String DBUSER=JOptionPane.showInputDialog(“Please input username:”);

String DBPASSWORD=JOptionPane.showInputDialog(“Please input password:”);

conn=DriverManager.getConnection(DBURL,DBUSER,DBPASSWORD);

JOptionPane.showMessageDialog(null,conn+”n数据库连接正常！“, “”,JOptionPane.PLAIN_MESSAGE);

 

Statement stmt=null;

String sql=”insert into student (Sname,Sage,Sdate) values(‘wanli’,20,’1991-03-14′)”;

stmt=conn.createStatement();

stmt.executeUpdate(sql);

stmt.close();

 

conn.close();

JOptionPane.showMessageDialog(null,”数据库关闭正常！“, “”,JOptionPane.PLAIN_MESSAGE);

}

}

 

 

此处就直接执行！

同理，删除和修改只要修改sql字符串即可。

 


数据库查询：

在查询的时候使用Statement类中的executeQuery()方法进行查询操作。看前面的表，此方法返回的是ResultSet接口。

看下面代码：

import javax.swing.*;

import java.sql.DriverManager;

import java.sql.Connection;

import java.sql.Statement;

import java.sql.ResultSet;

public class Database002

{

public static void main(String args[])throws Exception{

String DBURL=”jdbc:mysql://localhost:3306/testdata”;

String USER=JOptionPane.showInputDialog(“Please input username:”);

String PASS=JOptionPane.showInputDialog(“Please input password:”);

Connection conn=null;

conn=DriverManager.getConnection(DBURL,USER,PASS);

JOptionPane.showMessageDialog(null,conn+”n数据库连接正常！“,””,JOptionPane.PLAIN_MESSAGE);

Statement stmp=null;

ResultSet rel=null;

String sql=”select * from student”;

stmp=conn.createStatement();

rel=stmp.executeQuery(sql);

String S=”Sno     Sname     Sage     Sdatenn”;

while(rel.next()){

S=S+rel.getInt(“Sno”)+”     “;

S=S+rel.getString(“Sname”)+”     “;

S=S+rel.getInt(“SAge”)+”     “;

S=S+rel.getDate(“Sdate”)+’n';

}

JOptionPane.showMessageDialog(null,S,””,JOptionPane.PLAIN_MESSAGE);

conn.close();

JOptionPane.showMessageDialog(null,”数据库关闭正常！“,””,JOptionPane.PLAIN_MESSAGE);

}

}

很明显，使用ResultSet的next方法来移动指针，然后分别使用其getInt,getString等方法取得数据，参数里面的字符串必须是列名或者编号（列号）！


注：此处使用了“*”，实际开发中一般不使用“*”！
