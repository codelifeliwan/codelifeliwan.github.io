题目：
Stacking Boxes 

Background

Some concepts in Mathematics and Computer Science are simple in
one or two dimensions but become more complex when extended to
arbitrary dimensions. Consider solving differential equations in
several dimensions and analyzing the topology of an
n-dimensional hypercube. The former is much more complicated
than its one dimensional relative while the latter bears a
remarkable resemblance to its “lower-class” cousin.

The
Problem

n-dimensional boxes. You are to determine the longest

The
Input

The input consists of a series of box sequences. Each box
sequence begins with a line consisting of the the number of boxes
k in the sequence followed by the dimensionality of the
boxes, n (on the same line.)

This line is followed by k lines, one line per box with
the n measurements of each box on one line separated by one
or more spaces. The 103 - Stacking Boxes 解题报告” /> line in the sequence ( 103 - Stacking Boxes 解题报告” /> ) gives the measurements for the
103 - Stacking Boxes 解题报告” /> box.

There may be several box sequences in the input file. Your
program should process all of them and determine, for each
sequence, which of the k boxes determine the longest nesting
string and the length of that nesting string (the number of boxes
in the string).

In this problem the maximum dimensionality is 10 and the minimum
dimensionality is 1. The maximum number of boxes in a sequence is
30.

The
Output

For each box sequence in the input file, output the length of
the longest nesting string on one line followed on the next line by
a list of the boxes that comprise this string in order. The
“smallest” or “innermost” box of the nesting string should be
listed first, the next box (if there is one) should be listed
second, etc.

The boxes should be numbered according to the order in which
they appeared in the input file (first box is box 1, etc.).

If there is more than one longest nesting string then any one of
them can be output.

int number;
int maxn;
bool mycmp(int a,int b){
return a>b;
}
bool Judge(int m,int n,int dia){
for(int i=0;i<dia;i++){
if(box[m][i]<=box[n][i])
return false;
}
return true;
}
int dp(int m){
int &ans=vis[m];
if(ans>0)return ans;
ans=1;
for(int i=0;i<num;i++){
if(map[m][i]){
int tmp=dp(i)+1;
if(ans<tmp)ans=tmp;
}
}
return ans;
}

void dfs(int step,int p){

return;
}
for(int i=0;i<num;i++){
if(map[p][i]){
path[step+1]=i+1;

}
}
