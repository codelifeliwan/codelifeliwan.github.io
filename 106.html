UVA 146 – ID Codes 解题报告

Posted on 31 七月, 2011 by wanli — No Comments ↓

本题意思是求下一个排列，按照字典序排列的。

这里有一个函数：next_permutation:
函数是生成下一个字典序的排列，如果已经到了排列的最后一个，那么返回false，否则返回true。例如：
string s=”123″;
bool tmp=next_permutation(s.begin(),s.end());
那么现在tmp=true,s=”132″
如果现在
string s=”321″;
bool tmp=next_permutation(s.begin(),s.end());
那么现在tmp=false,s=”123″
那么这题就简单了吧。。。。。
