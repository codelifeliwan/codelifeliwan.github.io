JSP四种属性范围


四种范围分别为：

·pageContext

·request

·session

·application

 


属性的操作方法：

public void setAttribute(String name,Object obj)
设置属性name的值为obj。

public Object getAttribute(String name)
获得属性范围

public void removeAttribute(String name)
删除属性。

 


pageContext属性范围：

pageContest属性范围是单个页面的属性范围，是默认的属性范围，就是其定义的变量只能在本页使用。

 


request属性范围：

范围是在一次服务器跳转中有效，服务器跳转指的是浏览器地址栏地址不变的跳转，注意：超链接不属于服务器跳转，属于客户端跳转。

 


session范围：

在浏览器不关闭的情况下有效，也就是比request多了个超链接的范围（准确的说是改变地址栏地址）。

 


application属性范围：

在服务器不关闭的情况下，任何用户都能访问。
