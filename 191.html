Java常用类库–观察者模式

如果要想实现观察者模式，则要依靠java.util中提供的Observable类和Observer接口。

 


Observer接口：

方法：


 void



update(Observable o, Object arg)
          只要改变了 observable 对象就调用此方法。

其中第一个参数o是被观察的对象，第二个参数arg是被观察对象的改变的参数

Observable常用方法：

protected void setChanged()：用来标志某样东西已经变化

void notifyObservers(Object c)：参数是变化的属性，用来告诉观察者属性已经改变

void addObserver(Observer o)：用来说明o可以监视此对象

 

实例：


package ObserverTest;


import java.util.*;

 


class House extends Observable{

//表示能被观察的类


int price;


public House(int p){


price=p;


}


public void setPrice(int p){


if(p!=price){

//如果价格跟原来的价格不一样


super.setChanged();

//设置价格已经变化


super.notifyObservers(price);

//检测告诉观察者价格已经改变


price=p;


}


}


public String toString(){


return “价格为:”+price;


}


}

 


class Buyer implements Observer{


String name;


public Buyer(String s){


name=s;


}


public void update(Observable o,Object arg){

//被观察者一改变就调用此方法


System.out.println(“X+ “+o);


System.out.println(arg);


}


}

 


public class ObserverTest001 {


public static void main(String args[]){


House h=new House(10000);


Buyer b=new Buyer(“Hello”);


h.addObserver(b);


h.setPrice(10000);
//价格跟原来的价格一样,则不输出


System.out.println(“————————-“);


h.setPrice(9000);
//价格跟原来价格不一样


}


}

 


输出：


————————-


X+ 价格为:10000


10000
