 Java常用类库——StringBuffer类

StringBuffer本身也是操作字符串的，与String类不同，String类一旦声明不可改变，改变的只是其内存指向，而StringBuffer的内容是可以改变的。

StringBuffer是在java.lang中声明的。

常用方法：


apped：返回一个StringBuffer对象，参数可以为任意类型，加入内容。例如：


package StringBufferTest;

 


public class StringBufferTest001 {


public static void main(String args[]){


StringBuffer buf=new StringBuffer();


buf.append(“Hello”);


buf.append(” World”).append(“!!!n”);


System.out.println(buf);


}


}

输出：Hello World!!!

 

 

可使用方法传递：
