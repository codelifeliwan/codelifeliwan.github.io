Java基础（一）

Posted on 4 八月, 2011 by wanli — No Comments ↓

声明：此文假定读者已学会C/C++

常用输入输出：
System.out.print()输出不自动换行
System.out.println()输出自动换行
System.out.printf()和C相似
System.in.read()一次读取一个字节
Java数据类型：
与C++一样，Java有int,double等类型，但是不同的是，C++里面的bool型与java的不一样，java叫boolean，而不是写作bool。其各种数据表示范围与现在C的新标准差不多，号称Java能盛的下全宇宙的数据，也不过如此，做ACM大数运算的时候还是不行。。。。。
另外，Java里面的常量不是使用const，而是使用final关键字，而且final申请的变量可是事先不赋值，但是只能赋值一次。
在运算符方面，Java与C++基本相似，但是有一点需要说明：
在C++里面使用&&表示‘与’逻辑关系，在Java同样，但是大家知道，在C++里面，我有这样的式子：
int a=0;
int b=1;
if(a&&b++);
那么最后b的值为1，为什么呢，因为if语句执行前面的为假以后，后面的就不执行了。同理，Java也是这样。
但是在Java里面这个叫做短与，还有一种长与，只用一个符号表示：&，它执行完前面以后后面也是会执行的。
同理，有短或’||’和长或‘|’。
Java里面的强制转换和C一样，(int)a;
强调：Java没有指针类型和联合类型。
数组：
数组声明格式可以事先不说明大小，例如：int a[];或 int[] a;a=new int[5];
也可以说明大小例如：int a[]=new int[5];
同理，二维数组一样例如：
int a[][];a=new int[2][3];
int a[][]=new int[2][3];
这里使用数组名.length来表示数组大小。例如：
int a[]=new int[5];
那么a.length就等于5。
public class test {
public static void main(String args[]){
int a[]=new int[5];
System.out.println(a.length);
}
}
二维数组：int a[][]=new int[2][3];
那么a.length为2，a[0].length为3。
public class test {
public static void main(String args[]){
int a[][]=new int[2][3];
System.out.println(a.length);
System.out.println(a[0].length);
}
}
方法：
方法，在C++又叫函数，使用方法和C++差不多。
注意：static
方法只能调用static方法和使用非本方法自己定义的static变量！
字符串：
Java里面的字符串是一个String类，跟字符数组不一样，也要跟C++里面的string类区分开来。
有一点：
Java里面可以使用字符串常量相加的形式例如：
String S=”Hello “+”World!”;
而C++不行，具体原因在此不细说。
注意：不能使用字符数组风格：S[0]：错误！！！

使用方法来操纵字符串，常用方法如下：
int length()
boolean equals(Object obj)比较是否相等
char charAt(int index)返回index指定的字符
String substring(int begin)返回从begin到最后的子串
String substring(int begin,int end)返回从begin到end-1的子串
static String format(String format,…)返回format指定格式的字符串
The end！
