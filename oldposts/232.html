ubuntu 从unity到KDE

Posted on 4 四月, 2012 by wanli — No Comments ↓

要是有同学讨厌了新ubuntu的Unity桌面系统就试试这个吧。

同样是KDE，红旗的跟ubuntu的咋就差这么远呢？
说实话，原来老版本的gnome桌面系统用着还挺好，但是新版本的Unity桌面系统实在让人不敢恭维，ubuntu在极度弱化窗口的概念，但是我感觉这个不是很成功，新的桌面系统极度的不方便。
以前对于KDE说实话，有点恐惧，这个也是我一直没有愿意用KDE的原因，这种恐惧来源于红旗的那个系统，极度不好用。。。。。
昨天实在没办法了，又不想在用gnome了，就装了KDE。
安装步骤：
打开终端，输入：
sudo apt-get install kubuntu-desktop
然后提示下载选择Y
如果下载不成功的话先运行：
sudo apt-get update
然后在重新运行上面命令。
下载完成后选择kdm。
然后安装完成注销，在登录的时候选择KDE环境就OK了。
进入KDE后可以先卸载了Unity：

sudo apt-get  -y –auto-remove purge unity

sudo apt-get -y –auto-remove purge unity-commonp

sudo apt-get -y –auto-remove purge unity-lens*

sudo apt-get -y –auto-remove purge unity-services

sudo apt-get -y –auto-remove purge unity-asset-pool

然后KDE的中文包我一直没安好，求指点。
