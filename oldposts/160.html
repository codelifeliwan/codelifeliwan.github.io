补充：Java方法

Posted on 7 九月, 2011 by wanli — No Comments ↓

补充点容易被漏掉的东西：


可变参数：


看下面程序：

public class Main{

   
public static void main(String args[]){

      
fun(10,34,20);

   
}

   
public static void fun(int…arc){

      
for(int i=0;i<arc.length;i++){

          
System.out.println(arc[i]);

      
}

   
}

}


程序中fun方法定义如下：

   
public static void fun(int…arc)


这里加的省略号就是可变参数，其传的参数个数不限，arc当成数组处理即可。

 

foreach循环：


看下面程序：

public class Main{

   
public static void main(String args[]){

      
fun(10,34,20);

      
int a[]={10,34,78,12,847};

      
for(int x:a){

          
System.out.println(x);

      
}

   
}

   
public static void fun(int…arc){

      
for(int i=0;i<arc.length;i++){

          
System.out.println(arc[i]);

      
}

   
}

}

 


这里：

for(int x:a){

          
System.out.println(x);

      
}


意思就是把a数组中的元素循环输出，记住此用法。
