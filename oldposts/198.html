JSP内置对象–response

response对象表示HttpServletResponse对象。


response中的常用方法：

public void setHeader(String name,String value)：设置头信息。

public void sendRedirect(String Location)throws java.io.IOException：跳转到Location。

public void addCookie(Cookie cookie)：向客户端加入Cookie。

public void setContentType(String type)：设置内容的返回类型。

 


范例：


使用头设置头信息设置定时刷新：


<%@ page language=“java” import=“java.util.*” pageEncoding=“UTF-8″%>


<html>


  <head>


    <title>Response</title>


  </head>


  


  <body>


  
<%=(new java.util.Date()).toLocaleString() %><p>


  

<%response.setHeader(“refresh”,”1″); %>


  </body>


</html>

这时候就自动在页面上刷新时间（每秒一次）。

  </body>

和<jsp:forward>标签跳转的区别：

setHeader和sendRedirect是客户端跳转，而<jsp:forward>标签是客户端跳转。

Forward跳转是无条件跳转，跳转过后后面不再执行，而response跳转过后后面仍然会执行完毕。

 


Cookie：

Cookie是服务器端保留在客户端的一组数据。

Cookie类的常用方法：

public Cookie(String name,String value)：构造方法，传入名称和内容。

public String getName()：取得Cookie名字。

public void setValue(String S)：修改内容。

Public void setMaxAge(int expiry)：Cookie的最大保存时间。

 

取得Cookie使用request中的getCookies()方法。

Cookie c[]=request.getCookies();

 

范例:
