题目：

Problem B: Factstone Benchmark

Amtel
has announced that it will release a 128-bit computer chip by 2010,
a 256-bit computer by 2020, and so on, continuing its strategy of
doubling the word-size every ten years. (Amtel released a 64-bit
computer in 2000, a 32-bit computer in 1990, a 16-bit computer in
1980, an 8-bit computer in 1970, and a 4-bit computer, its first,
in 1960.)

10916 - Factstone Benchmark 解题报告” TITLE=”UVA 10916 - Factstone Benchmark 解题报告” />

Amtel will use a new benchmark –
the Factstone - to
advertise the vastly improved capacity of its new chips.
The Factstonerating is defined to be the
largest integer n such
that n! can be represented
as an unsigned integer in a computer word.

Given a year 1960 ≤ y ≤ 2160, what will be
the Factstone rating of
Amtel’s most recently released chip?

There are several test cases. For each test case, there is one line
of input containing y. A line containing 0
follows the last test case. For each test case, output a line
giving the Factstone rating.

Sample Input

1960 

1981 

0

Output for Sample Input

3 

8

代码：

#include<iostream>

#include<fstream>

#include<cmath>

using namespace std;







int main()

{

// freopen("test.txt","r",stdin);

 double year;

 while(cin>>year&&year)

 {

 double weishu=(pow(2,(int)(year-1960)/10))*4;

 int x;

 for(x=1;;x++)

 {

 weishu=weishu-(log10(x)/log10(2));

 if(weishu<0)break;

 }

 cout<<x-1<<endl;

 }

 return 0;

}
