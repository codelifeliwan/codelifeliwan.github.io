JSP内置对象-request

request是最常用的内置对象，表示javax.servlet.http.HttpServletRequest对象。

常用方法：

getParameter(String name)：取得name的值。

setCharacterEncoding(String env)：设置编码环境 例如：request.setCharacterEncoding(“GBK”);

 


get、post提交：

例如：


<form action=“response001.jsp” method=“post” name=“res”>

那么这里的method有两个取值：get和post。

get提交在浏览器地址栏里面小时所有提交信息，而post则不显示。

get提交提交内容有限。而post本身不受长度限制。

 

除了可以使用表单传递参数以外，还可以使用重写URL方法：

<a href=”URL?参数1=值1&参数2=值2&…”>…</a>

在URL页面里面可以使用request接收参数1、参数2、…

表单中有一种隐藏对象，本身不会显示，但是会跟着表单一起提交上去。

<input type=”hidden” name=”name” value=”value”…>

 


getParameterValues:

如果想用checkbox接收多个参数，则不能使用getParameter()方法，使用

getParameterValues(String name)方法接收：

String S[]=request.getParameterValues(“name”);

 


getParameterNames:

如果想取得所有表单提交参数的名称，则使用getParameterNames()方法。

要引用java.util.*

Enumeration enu=request.getParameterNames();

 


getRemoteAddr:

取得客户端IP地址使用getRemoteAddr()方法。

String ip=request.getRemoteAddr();

 


RequestURI:

返回发出请求的客户端地址。

Stringn S=request.RequestURI();

 


getSession:

获得session

HttpSession getSession();
