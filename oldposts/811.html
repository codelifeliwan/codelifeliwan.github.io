从头开始学shell编程

第一个shell程序

开始，我们写第一个shell程序，也就是传说中的helloworld程序，我们新建一个文本文档1.sh然后编辑如下：

? Download 1.sh 
1
2

#!/bin/sh
echo "Hello World!"

保存好后退出，更改其权限：chmod u+x 1.sh，然后直接执行./1.sh，这时候就看见屏幕上打印出来了Hello World!了。

让我们来解释一下上面两行代码，第一行为：
#!/bin/sh
在shell编程中，#后面的东西是注释，但是这一行除外，#后面很上!表示指定解释程序，这里说明此次shell的解释程序是/bin/sh，大多数/bin/sh是一个链接，链接到/bin/bash，当然我们也可以在第一行写上#!/bin/bash，甚至也可以不写，不写的话运行的时候要指定解释程序比如运行的时候需要这样：
sh 2.sh
第二行是一个打印命令，echo是回显，在这里就是将要回显的内容写道终端上。

变量

在shell中，变量有三种，分为本地变量，环境变量和位置参数。变量的基本操作就是赋值和替换，位置参数是只读变量，不可进行以上操作。
变量不许要声明可以直接使用。

变量的赋值格式如下：
variable=value
需要说明的是：
1、等号的两边可以有空格，不影响操作。
2、如果值包含空格，则必须用双引号包起来。
3、变量名规则和C语言一样。
代码如下：

#!/bin/sh
 
val1=bianliang1
echo $val1
val2="bian liang 2"
echo $val2
echo ${val2}

其中，在变量名前加$是取变量值的意思。
另外，我们也可以在一个变量的赋值过程中使用另一个变量：

? Download 1.sh 
1
2
3
4
5
6

#!/bin/sh
 
val1=bianliang1
echo $val1
val2="this is bianliang2 not $val1"
echo $val2

清除变量的值用unset如：
unset val1
变量的赋值在其他的情况下也可以使用，但是需要加上${}，注意中间无空格。
在此赋值模式下有如下几种方式：
1、var=value :将value值赋值给var
2、var+value :对已经赋值的var，重设其值，对未赋值的var无作用
3、var?value或var:?valur :对未赋值的var，显示系统错误信息
4、var:=value :对未赋值的var，将value值赋给它，对已经赋值的var无用
5、var:-value :对未赋值的var，将value赋给它，但是不存储在var中

echo "this is bianliang ---${val3:=3}---"
echo $val3
echo "this is bianliang ---${val4:-4}---"
echo $val4
echo "bianliang ${val3+$val2}"

也可以把变量设置为只读变量，只需要使用readonly命令即可，只读变量不可重新赋值或者修改等：
readonly val1
在终端找中使用readonly命令可以查看系统中的所有只读变量。

环境变量是系统中定义的变量，用来表示系统中程序的执行路径等，在系统中我们使用env命令可以查看全部的环境变量，如果想查看哪一项只需要打印出来即可，比如我们想查看环境变量PATH的值我们只需要：
echo $PATH
当然我们也可以自己定义环境变量，然后手工引入，环境变量名一般是大写，手工引入的时候使用export命令比如:

#!/bin/sh
 
MY_PATH=/home/test
export MY_PATH

这样，环境变量中就会有MY_PATH这一项，其值为/home/test
清除环境变量还是使用unset命令。
其中，有几个比较重要的环境变量在这里就不多说了。

关于shell的变量类型，在这里只能说，shell脚本的变量是无类型的，更确切地说shell脚本只支持整型和字符型，若一个字符串里面只有数字那么shell就将其当作整型否则就是字符型，字符型变量也可以进行运算，不过其初始值是0。

位置参数是一种特殊的环境变量，用于从命令行向shell脚本传递参数，$0表示shell脚本本身，$1表示第一个参数，$2表示第二个参数等，从${10}开始要使用花括号括起来。
另外还有几个比较特殊的：
$# ：表示传递到脚本的参数数量
$*或$@ ：表示传递到脚本的所有参数
$$ ：表示脚本运行的进程号
$? ：表示命令的退出状态,0表示正常退出

#!/bin/sh
 
echo "---the number of element is $#---"
echo "---elemets are $*---"
echo "--elements are $@---"
echo "---the pid is $$---"
echo "--#0=$0---"
echo "--#1=$1---"
echo "--#2=$2---"
echo "--#3=$3---"
echo "--#4=$4---"
echo "--#5=$5---"
echo "---exit code is $?---"

我们执行程序如下：./1.sh a b c
