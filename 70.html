63-看你的Webcam是否工作

打开一个Terminal，输入“gstreamer-properties”，打开“Video”标签，点击“测试”按钮，会出现一个窗口显示你的Webcam捕捉到的东西。如果出现错误，在“插件”下拉框里面选择“Video
for Linux(v4l)”。如果依然提示错误，可能你的Webcam就和Ubuntu不兼容了。

 

补充，如果Webcam可以测试成功，所有使用它的程序都可以正常使用。

 



64-使用Firefox2

 

Ubuntu8.04使用Firefox3，它带来了一些非常新颖而且有用的功能。但是，有些人可能对于Firefox3缓慢的启动速度不满意，喜欢降级到Firefox2。你可以在“新立得”里面搜索firefox-2里面搜索并安装Firefox
2。安装完成以后，可以在命令行里面输入firefox-2或者建立一个桌面快捷方式启动Firefox 2.

 

如果想全部使用Firefox2打开网络链接（譬如Envoluation里面的连接），可以在GNOME菜单“系统”-》“系统首选项”-》“缺省应用程序”里面，选择“互联网”变迁，在“浏览器”下拉框里面，选择“其他”，然后输入命令“firefox-2
%s”就可以了。

 

注意，Firefox2的插件可能工作不正常。如果你想使用插件，最好使用Firefox
3，这时只要输入“firefox”命令就可以了。

 



65-安装可能需要的所有多媒体解码器

Ubuntu会在你需要播放一个多媒体文件时候，自动安装解码器。现在的问题是，如果你必须连接到互联网上面，才能工作，如果你在没有网络连接的地方，想播放一个没有安装解码器的多媒体文件的时候，就会出现问题。因此，你可能想移除安装所有的多媒体解码器。

 

打开“新立得”，搜索“gstreamer”，找到下面的软件：

GStreamer extra plugins

GStreamer ffmpeg video plugin

Ubuntu restricted extras

GStreamer plugins for mms, wavpack, quicktime, musepack

GStreamer plugins for aac, xvid, mpeg2, faad

GStreamer fluendo MPEG2 demuxing plugin

 

安装就可以了。安装过程中，可能会弹出几个版权协议对话框。

 

如果想播放DVD电影，你必须完成上面的步骤。然后关闭“新立得”，打开一个Terminal，输入：

$ sudo /usr/share/doc/libdvdread3/install-css.sh

67-一键开启一个Terminal

 

我不知道你的情况，但是我有50%的时间是花费在命令行里面，所以能够快速开启一个Terminal无疑是一件有意义的事情。

 

建立开启Terminal，可以在”系统“-》”系统首选项“-》”快捷键“里面，找到”运行一个终端（Run a
Termial)“，去掉”禁用“，输入Ctrl+Alt+t就可以了。当然你可以定义其他的快捷键，如果你有一个“Windows”键（就是键盘上那个有Windows
logo的键）也可以使用。但是尽量不要和其他的快捷键冲突，譬如定义了Ctrl+t，就和Firefox建立新标签页的快捷键冲突了。

70-立即关闭所有的网络连接

 



你是否有的时候觉得电脑被入侵了？或者你刚刚点击“发送”按钮，给老板发了一封邮件但是立刻就会后悔了？这个时候，你只需在“通知区域”里面的NetworkManager图标上面点击右键，去掉勾选“连接网络”就可以关闭网络连接了。NetworkManager可能提示一些提示信息。回复网络连接，只需重新勾选就可以了。

 



71-从Ubuntu桌面发布Blog

 

你可能不是一个Blogger，但是相信我，“写一篇Blog并把发表到世界上是一个非常棒的注意。GNOME的开发者，似乎意识到这一点，于是开发了从Ubuntu桌面快速发表Blog的软件。

 

从“新立得”里面找到gnome-blog，并且安装这个软件。安装完成以后，右键点击GNOME面板空白处，选择“添加到面板。。”，从列表里面选择“Blog
Entry Poster”即可。

 

Gnome-blog支持非常多的Blog供应商，譬如Blogger.com、Advogato、Live
Jouranl等，另外还可以你还可以自己配置，让他可以发布到你的主机上面安装的WordPress、MovableType、Pyblosxon等Blog软件。

 

第一次运行，这个软件会提示具体配置Blog。你可以从“Blog
Type”里面选择你的Blog类型，输入用户名和密码，然后选择“lookup
blogs”按钮，查看返回的系统信息是否正确，并且可以显示以前发布的Blog的列表。

 

发布Blog，只需点击面板上面的applet按钮，输入标题和内容，点击“Post Entry”按钮就可以了，同时支持图片拖放。

 



72-智能选择部分文件

想象一下下面的场景：

你在一个工作组里面工作。在你的“文档”文件夹里面存储这这个项目的文件。这些项目文件包括图片文件、Writer文档、SpreadSheets文档等。

 

现在你的Boss需要几个文件，要求你在很短的时间以内把一下文件找到并送到他那里。

 

现在的问题是，文件太多了，甚至用扩展名排序对于查找文件也帮助不大。

 

如果文件名里面都保护项目的名称，你可以使用Nautilus的“Search
Pattern”功能。譬如项目的名称试“Falken”，你可以打开Nautilus的“编辑”菜单，找到“Search
Pattern”，输入

*Falken*

 

 

*是一个通配符，上面的语句匹配所有文件名包含Falken的文件，并且这些文件现在就会在Natilus里面被选中了，现在可以把他们拖进Email软件，发送给你的Boss了。

 



73-暂时禁用一个用户账号

 

如果你的看过“秘籍50”，里面论述了如何让Ubuntu建立儿童账号，防止他们访问“儿童不宜”的Internet网站。现在，你可以有的时候不希望他们上网，譬如希望他们做一些其他的事情。

 

下面的内容，将帮助你暂时禁用儿童账号：

Sudo password –l username

其中username就是用户名。

 

如果想启用“已经暂时禁用的账号”，只需：

Sudo password –u username
