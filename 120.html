关于ubuntu下java环境配置及其运行

Posted on 4 八月, 2011 by wanli — No Comments ↓

其实不需要怎么配置，直接到ubuntu软件中心里面搜jre就行了，安装上。

在终端输入：sudo apt-get install openjdk-6-jdk
再在ubuntu软件中心里面搜索eclipse下载安装即可。
不安装eclipse也行，不过运行什么的都要在终端了。。。。。
下面就运行试试：
1.直接终端运行：
在桌面建立一个文本文档命名为test.java，使用其他编辑器也行。
在里面输入：
public class test{//记住这里类名必须和文档名一致
      public
static void main(String args[]){
     
    for(int
i=0;i<10;i++){
     
     
  System.out.println(“Hello World”);
     
    }
   
  }
  }

保存好后在终端输入：
cd ~Desktop
输入javac test.java
将在桌面上出现一个test.class文件。
再输入java test
将出来结果如下：

此为正确结果。
使用eclipse如下：
打开eclipse，点击File->New->Java
Project.
选择路径，输入工程名（test为例）。
在工程名上点击右键：New->class
输入test回车
输入
public class test {
public static void main(String args[]){
System.out.println(“Hello World!”);
}
}
点击左上方运行：
